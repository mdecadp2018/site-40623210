*介紹 V-rep 功能
(將繪圖軟體的零組件輸入V-rep進行動態模擬)

1. 將組和完成圖檔, 以 STL 格式轉出。

2. 將 STL 組件輸入 V-rep 後, 將組件根據零件運動軸進行設定。

3. 以 Lua 嵌入或 Python remote API 方式驅動各致動器進行模擬。
