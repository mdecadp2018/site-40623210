Onshape Assessment10

10-3-Assessment

10-3-Assessment (Answers)

1. How does the “Hole” feature save time?

“孔”功能如何節省時間？

Since most holes have standard sizes, the Hole feature has presets geometry to allow the user to just select from a library of options, instead of having to sketch the profile of the hole. In addition, it just references existing sketch points, thus saving time duringhole placement as well.

由於大多數孔都具有標準尺寸，因此孔特徵具有預設幾何圖形，以允許用戶僅從選項庫中進行選擇，而不必繪製孔的輪廓。 此外，它僅引用現有的草圖點，從而節省了在鑽孔放置期間的時間。

2. What does the “offset” option do in the Spur Gear FeatureScript feature?

“偏移”選項在正齒輪功能特徵中的作用是什麼？

It allows the gear to be “clocked” so adjacent gears can be perfectly meshed.

它允許齒輪“定時”，因此相鄰的齒輪可以完美嚙合。

3. True or False: Onshape can directly import a Solidworks Pack n Go file.

對錯：Onshape 可以直接在文件中導入Solidworks Pack。

True.

對

4. Name two of the four “Direct Editing” features in Onshape.

在 Onshape 中命名四個 “ 直接編輯 ” 功能中的兩個。

a. Modify Fillet

修改圓角

b. Delete Face

刪除面

c. Move Face

移動面

d. Replace Face

替換面

5. List 3 different capabilities that can be found in the App store.

列出可在App Store中找到的3種不同功能。

Rendering, CAM, BOM, Assembly Explode, FEA/CFD Simulation, 3D Printing, hardware libraries, etc.

渲染，CAM，BOM，裝配爆炸，FEA / CFD模擬，3D打印，硬件庫等
